# House-price-prediction

## Project Overview

This project focuses on building a Machine Learning model to predict residential property prices for a real estate company operating across multiple cities.

The goal is to automate the house pricing process by developing an intelligent system that estimates property prices based on various features such as location, physical attributes, and condition.

## Problem Statement

This real estate company handles thousands of property transactions annually and requires an automated pricing system.

The objective of this project is to:

- Analyze historical housing data
- Build a predictive model using Machine Learning
- Accurately estimate house prices

## Technologies Used
Python 
NumPy
Pandas
Matplotlib / Seaborn
Scikit-learn

## Results

The model successfully predicts house prices with reasonable accuracy and can assist in automating the pricing process for real estate businesses.
